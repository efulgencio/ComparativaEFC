# ComparativaEFC
Comparativa fotos por filtro aplicado y brillo.



Un editor de fotos minimalista en un solo archivo de SwiftUI. Diseñado para aprender a usar Core Image y el procesamiento de fotos de forma acumulativa (filtros + brillo) sin perder calidad.

✨ ¿Qué hace?
Carga Fotos: Usa el selector nativo de iOS para elegir imágenes de tu galería.

Edición No Destructiva: Puedes aplicar filtros de color y ajustar el brillo por separado; los cambios se suman sin borrarse entre sí.

Slider Vertical: Un control de brillo personalizado que muestra el porcentaje en tiempo real.

Comparativa: Un carrusel inferior para guardar tus versiones favoritas y comparar cómo quedan.
